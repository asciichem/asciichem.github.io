Every block on this page is rendered at build time by invoking the
`asciichem` gem. Source on the left; MathML output on the right. If you
view the page source, you'll see the same is what you'd type.

```asciichem
CO_2

CH_4
```

## Isotopes — the headline semantic fix

```asciichem

^131I
^238U
^3H
```

The prefix isotope binds to the atom. See [Atoms](/syntax/atoms/) for
the full story.

## Verify locally

```bash
gem install asciichem
asciichem convert -i "2H_2 + O_2 -> 2H_2O" -t mathml
asciichem convert -i "2H_2 + O_2 -> 2H_2O" -t latex
asciichem roundtrip -i "2H_2 + O_2 -> 2H_2O"
```
